# Projeto Aplicado I - FIFA World Cup
Análise exploratória de dados do maior torneio de seleções do Mundo de 1930 até 2014.

# Introdução
A Copa do Mundo da FIFA é o torneio mais prestigiado no futebol internacional. É organizada pela FIFA (Federação Internacional de Futebol) e ocorre a cada quatro anos, atraindo milhões de espectadores de todo o mundo. O torneio apresenta equipes de todo o mundo competindo pelo título de campeão mundial.
	
Com isso realizaremos uma análise sobre o dataset oferecido pela FIFA sobre todos os jogos ocorridos até a Copa do Mundo de 2014, o objetivo aqui é conhecer as principais estatísticas do torneio e descobrir e tirar insights interessantes e curiosos através dos dados dispostos.
